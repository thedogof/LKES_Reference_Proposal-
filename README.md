# 龍科國小校舍規劃 Web App（工程概念評估版）

本專案為**可直接展示的靜態 Web App**，重點包含：

- 全球 8 大校園案例互動比對
- 龍科在地化策略摘要
- 工程概念評估器（參數化）：
  - 抗震（Cs、基底剪力 V）
  - 日照效率
  - 風場與換氣率（ACH）
  - 熱舒適指數
- 方案 A/B 比較表
- 校園圍牆設計提案（美觀 × 安全 × 在地）

## 本機啟動

```bash
python3 -m http.server 4173
```

開啟：`http://127.0.0.1:4173`

## 注意事項

- 本工具定位為「前期規劃/方案比較」用途。
- 結構、機電、熱流等最終設計仍需專業顧問依現行法規與簽證流程完成。

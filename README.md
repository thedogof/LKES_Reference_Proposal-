# 龍科國小校舍規劃 Web App（旗艦級參考版）

這個版本重點是**更完整參考性 + 更高質感 UI/UX + 更直覺互動流程**：

- 八大案例完整卡（每案策略＋指標＋多張參考圖片）
- 圖片可點擊放大（Lightbox）
- 工程前期評估器（抗震/日照/風場/熱舒適）
- 方案 A/B 儲存比較 + 匯出摘要
- 圍牆設計提案（美學 × 安全 × 在地）
- 響應式布局、滾動動畫、回頂按鈕、頂部快捷導覽

## 本機啟動

```bash
python3 -m http.server 4173
```

打開：`http://127.0.0.1:4173`

## 注意

- 本工具為前期規劃與溝通用途，不可取代正式結構/機電/法規簽證。
- 若進入下一階段，建議接 BIM、CFD 與法規審查流程做精算驗證。
